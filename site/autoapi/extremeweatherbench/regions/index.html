    CenteredRegion(latitude, longitude, bounding_box_degrees)

    Bases: Region

    A region defined by a center point and a bounding box.

    bounding_box_degrees is the width (length) of one or all sides, not half size; e.g., bounding_box_degrees=10.0 means a 10 degree by 10 degree box around the center point.

    RegionSubsetter 

    RegionSubsetter(
    region, method="intersects", percent_threshold=0.5
)

    A utility class for subsetting ExtremeWeatherBench objects by region.

    Attributes:

    • region (Region) –

      The region to subset to. Can be a Region object or a dictionary of bounds with keys "latitude_min", "latitude_max", "longitude_min", and "longitude_max".

    • method (str) –

      The method to use for subsetting. Options: - "intersects": Include cases where ANY part of a case intersects region - "percent": Include cases where percent of case area overlaps with region. - "all": Only include cases where entirety of a case is within region

    • percent_threshold (float) –

      Threshold for percent overlap (0.0 to 1.0)

    Parameters:

    • region (Union[Region, Mapping[str, float]]) –

      The region to subset to. Can be a Region object or a dictionary of bounds with keys "latitude_min", "latitude_max", "longitude_min", and "longitude_max".

    • method (Literal['intersects', 'percent', 'all'], default: 'intersects' ) –

      The method to use for subsetting. Options: - "intersects": Include cases where ANY part of a case intersects region - "percent": Include cases where percent of case area overlaps with region - "all": Only include cases where entirety of a case is within region

    • percent_threshold (float, default: 0.5 ) –

      Threshold for percent overlap (0.0 to 1.0)

    Methods:

    subset_results_to_region 

    subset_results_to_region(
    region, results_df, case_collection
)

    Subset results DataFrame by region using case_id_number.

    This is a convenience function that creates a RegionSubsetter and applies it to the results DataFrame that is output from ExtremeWeatherBench.run().

    Parameters:

    • region (RegionSubsetter) –

      The region to subset to. Can be a Region object or a dictionary of bounds with keys "latitude_min", "latitude_max", "longitude_min", and "longitude_max".

    • results_df (DataFrame) –

      DataFrame with results from ExtremeWeatherBench.run()

    • case_collection (IndividualCaseCollection) –

      The original case collection to determine which case_id_numbers correspond to cases in the region

    Returns:

    • DataFrame –

      Subset DataFrame containing only results for cases in the region
