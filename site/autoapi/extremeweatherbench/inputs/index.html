    EvaluationObject dataclass 

    EvaluationObject(event_type, metric_list, target, forecast)

    A class to store the evaluation object for a forecast and target pairing.

    A EvaluationObject is an evaluation object which contains a forecast, target, and metrics to evaluate. The evaluation is a set of all metrics, target variables, and forecast variables.

    Multiple EvaluationObjects can be used to evaluate a single event type. This is useful for evaluating distinct targets or metrics with unique variables to evaluate.

    Attributes:

    • event_type (str) –

      The event type to evaluate.

    • metric_list (Sequence[BaseMetric]) –

      A list of BaseMetric objects.

    • target (TargetBase) –

      A TargetBase object.

    • forecast (ForecastBase) –

      A ForecastBase object.

    open_kerchunk_reference 

    open_kerchunk_reference(
    forecast_dir,
    storage_options={
        "remote_protocol": "s3",
        "remote_options": {"anon": True},
    },
    chunks="auto",
)

    Open a dataset from a kerchunked reference file in parquet or json format. This has been built primarily for the CIRA MLWP S3 bucket's data (https://registry.opendata.aws/aiwp/), but can work with other data in the future. Currently only supports CIRA data unless schema is identical to the CIRA schema.

    Parameters:

    • forecast_dir (str) –

      The path to the kerchunked reference file.

    • storage_options (dict, default: {'remote_protocol': 's3', 'remote_options': {'anon': True}} ) –

      The storage options to use.

    • chunks (Union[dict, str], default: 'auto' ) –

      The chunks to use; defaults to "auto".

    Returns:

    • Dataset –

      The opened dataset.
